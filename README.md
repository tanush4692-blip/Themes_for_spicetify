# Nothing Sleek ⬛⬜
A clean, minimal, highly polished and Nothing(R) inspiered custom theme for [Spicetify](https://spicetify.app/). 

## ⚠️ Important Cautions Before Installing

• **Spicetify Required:** You must have Spicetify CLI installed before applying this theme.

• **Spotify Version:** Spicetify does **not** support the Windows Store version of Spotify. You must use the desktop client downloaded directly from Spotify's website.

• **Updates Can Break Themes:** Spotify pushes frequent updates that can occasionally break Spicetify themes. If your Spotify looks broken after an update, open PowerShell and run `spicetify upgrade` followed by `spicetify restore backup apply`.

## 📥 Installation

1. Download the Files
Download or clone this repository to your computer. You will need the `Nothing_Sleek` folder and the included font file.

2. Install the Font
For the theme to render correctly, Spicetify needs access to the custom font.
3. Locate the font file included in this repository.

4. Drag and drop the font file directly into your root `.spicetify` folder (typically located at `C:\Users\YOUR_USERNAME\.spicetify`). 
	*(Note: You may also want to install the font to your operating system by double-clicking it and selecting "Install", just to be safe).*

5. Move the Theme Folder

6. Locate the `Nothing_Sleek` folder you downloaded.

7. Move the entire `Nothing_Sleek` folder into your Spicetify **Themes** directory.

   	**Windows:** `C:\Users\YOUR_USERNAME\AppData\Roaming\spicetify\Themes\`
   	**Linux/macOS:** `~/.spicetify/Themes/`
   
	*(Tip: You can easily find your Spicetify folder by opening your terminal and running `spicetify -c` to see exactly where your config file is 			located).*

## 🚀 Applying the Theme
Once the files are in the right place, you need to tell Spicetify to use them.

-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_
  Windows(Powershell/Terminal)
_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-

1. Open **PowerShell** (Windows) or **Terminal** (macOS/Linux).

2. Run the following command to set the theme:powershell
	spicetify config current_theme Nothing_Sleek

3. Apply the changes and restart Spotify by running:

	spicetify apply

-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_
  Linux & macOS (Terminal)
_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-
Open your Terminal.

2. Run the following commands one by one:Bash
   spicetify config current_theme Nothing_Sleek

3. Apply the changes and restart Spotify by running:
   spicetify apply
